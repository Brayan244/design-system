## Row example
With the `horizontal` direction and the `flex: 1` property, rows can be created with children of the same size and occupy 100% of the width.

<Preview>
  <Story name="Row example">
    <Spacing
      direction="horizontal"
        sizeHorizontal="M"
        
        flex='1'
        isResponsive
      >
      <Input
        label="Nombre"
        id="name"
        value="Mary Elizabeth"
        onChange={action('onChange')}
      />
      <Input
        label="Apellido"
        id="lastname"
        value="Winstead"
        onChange={action('onChange')}
      />
    </Spacing>
  </Story>
</Preview>

## Spacing inside Spacing
You can use the Spacing component inside another Spacing component to create layouts.
The first Spacing has a vertical direction, the inner Spacing have a horizontal direction.

<Preview>
  <Story name="Spacing inside spacing">
    <Spacing direction="vertical" sizeVertical="L">
      <Spacing
      direction="horizontal"
        sizeHorizontal="M"
        
        flex='1'
        isResponsive

      <Input
        label="Nombre"
        id="name"
        value="Mary Elizabeth"
